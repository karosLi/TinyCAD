================================================================================
    MICROSOFT 基础类库: TinyCAD 项目概述
===============================================================================

应用程序向导已为您创建此 TinyCAD 应用程序。此应用程序不仅演示使用 Microsoft 基础类的基本知识，而且可作为编写应用程序的起点。

此文件包含组成 TinyCAD 应用程序的各个文件的内容摘要。

TinyCAD.vcproj
    这是使用应用程序向导生成的 VC++ 项目的主项目文件。
    它包含有关生成文件的 Visual C++ 版本的信息，以及有关使用应用程序向导选择的平台、配置和项目功能的信息。

TinyCAD.h
    这是应用程序的主要头文件。它包括其他项目特定的头文件(包括 Resource.h)，并声明 CTinyCADApp 应用程序类。

TinyCAD.cpp
    这是包含应用程序类 CTinyCADApp 的主要应用程序源文件。

TinyCAD.rc
    这是程序使用的所有 Microsoft Windows 资源的列表。它包括 RES 子目录中存储的图标、位图和光标。此文件可以直接在 Microsoft Visual C++ 中进行编辑。项目资源位于 2052 中。

res\TinyCAD.ico
    这是用作应用程序图标的图标文件。此图标包括在主要资源文件 TinyCAD.rc 中。

res\TinyCAD.rc2
    此文件包含不是由 Microsoft Visual C++ 编辑的资源。您应该将不可由资源编辑器编辑的所有资源放在此文件中。

TinyCAD.reg
    这是一个示例 .reg 文件，它显示了框架将为您设置的注册设置的种类。可以将它用作
    将与您的应用程序一起使用的 .reg 文件，或者索性将它删除，而采用默认的 RegisterShellFileTypes 注册。

TinyCAD.idl
    此文件包含应用程序类型库的接口描述语言源代码。

/////////////////////////////////////////////////////////////////////////////

对于主框架窗口:
    项目包含标准 MFC 界面。

TinyCADMainFrm.h, TinyCADMainFrm.cpp
    这些文件包含框架类 CTinyCADMainFrame，该类派生自
    CMDIFrameWnd 并控制所有 MDI 框架功能。

res\Toolbar.bmp
    此位图文件用于为工具栏创建平铺图像。
    初始工具栏和状态栏是在 CTinyCADMainFrame 类中构造的。使用资源编辑器编辑此工具栏位图，并更新 TinyCAD.rc 中的 IDR_MAINFRAME TOOLBAR 数组以添加工具栏按钮。

res\IToolbar.bmp
    当服务器应用程序在另一个容器内部就地激活时，此位图文件用于为工具栏创建平铺图像。此工具栏是在 CTinyCADInPlaceFrame 类中构造的。此位图与 res\Toolbar.bmp 中的位图类似，不同之处是它删除了许多非服务器命令。

/////////////////////////////////////////////////////////////////////////////

对于子框架窗口:

ChildFrm.h, ChildFrm.cpp
    这些文件定义并实现 CTinyCADChildFrame 类，该类支持 MDI 应用程序中的子窗口。

/////////////////////////////////////////////////////////////////////////////

应用程序向导创建一个文档类型和一个视图:

TinyCADDoc.h，TinyCADDoc.cpp - 文档
    这些文件包含 CTinyCADDoc 类。编辑这些文件可以添加特殊文档数据并可实现文件保存和加载(通过 CTinyCADDoc::Serialize)。
    文档将具有下列字符串:
        文件扩展名:TCD
        文件类型 ID:TinyCAD.Document
        主框架标题:TinyCAD
        文档类型名称:TinyCAD
        筛选器名:TinyCAD 文档 (*.TCD)
        文件的新简称:TinyCAD
        文件类型全名:TinyCAD.Document

TinyCADView.h，TinyCADView.cpp - 文档的视图
    这些文件包含 CTinyCADView 类。
    CTinyCADView 对象用于查看 CTinyCADDoc 对象。

res\TinyCADDoc.ico
    这是图标文件，它用作 CTinyCADDoc 类的 MDI 子窗口的图标。此图标包括在主要资源文件 TinyCAD.rc 中。


/////////////////////////////////////////////////////////////////////////////

应用程序向导还创建了特定于 OLE 的类

TinyCADCntrItem.h, TinyCADCntrItem.cpp
    这些文件包含 CTinyCADCntrItem 类。此类用于操作 OLE 对象。OLE 对象通常由 CTinyCADView 类显示并作为 CTinyCADDoc 类的一部分序列化。
    程序支持在其框架中包含活动文档。

TinyCADSrvrItem.h, TinyCADSrvrItem.cpp
    这些文件包含 CTinyCADSrvrItem。此类用于将 CTinyCADDoc 类连接到 OLE 系统，并可以选择提供指向文档的链接。
    项目支持创建和管理活动文档。

TinyCADIpFrame.h, TinyCADIpFrame.cpp
    这些文件包含 CTinyCADInPlaceFrame。此类派生自 COleIPFrameWnd，并在就地激活期间控制所有框架功能。

    项目支持复合文件。复合文件格式将包含一个或多个自动化对象的文档存储到一个文件中，并仍允许访问其中的各个对象。

帮助支持：

hlp\TinyCAD.hhp
    此文件是帮助项目文件。它包含将帮助文件编译为 .chm 文件所需的数据。

hlp\TinyCAD.hhc
    此文件列出帮助项目的内容。

hlp\TinyCAD.hhk
    此文件包含帮助主题的索引。

hlp\afxcore.htm
    此文件包含标准 MFC 命令和屏幕对象的标准帮助主题。将您自己的帮助主题添加到此文件中。

hlp\afxprint.htm
    此文件包含打印命令的帮助主题。

makehtmlhelp.bat
    此文件由生成系统用来编译帮助文件。

hlp\Images\*.gif
    这些是 Microsoft 基础类库标准命令的标准帮助文件主题所需的位图文件。


/////////////////////////////////////////////////////////////////////////////

其他功能:

ActiveX 控件
    应用程序包括对使用 ActiveX 控件的支持。

打印及打印预览支持
    应用程序向导已通过从 MFC 库调用 CView 类中的成员函数，生成了用于处理打印、打印设置和打印预览命令的代码。

拆分窗口
    应用程序向导已为应用程序文档添加了拆分窗口支持。

/////////////////////////////////////////////////////////////////////////////

其他标准文件:

StdAfx.h, StdAfx.cpp
    这些文件用于生成名为 TinyCAD.pch 的预编译头(PCH)文件和名为 StdAfx.obj 的预编译类型文件。

Resource.h
    这是标准头文件，它定义新资源 ID。
    Microsoft Visual C++ 将读取并更新此文件。

TinyCAD.manifest
	Windows XP 使用应用程序清单文件描述应用程序	对特定版本并行程序集的依赖性。加载程序使用此	信息从程序集缓存加载相应程序集或	从应用程序加载私有信息。应用程序清单可能作为	与应用程序可执行文件安装在同一文件夹中的外部 .manifest 文件包括在内以便重新发布，	也可能以资源的形式包括在该可执行文件中。

其他注释:

应用程序向导使用“TODO:”指示应添加或自定义的源代码部分。

如果应用程序在共享 DLL 中使用 MFC,则将需要重新发布 MFC DLL。如果应用程序所用与操作系统的区域设置不同，则也将必须重新发布对应的本地化资源 MFC90XXX.DLL。
有关这两个主题的详细信息，请参阅 MSDN 文档中有关重新发布 Visual C++ 应用程序的部分。
